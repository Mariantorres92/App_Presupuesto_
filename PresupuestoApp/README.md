# PresupuestoApp

# 💰 Aplicación de Presupuesto Personal

Proyecto desarrollado para el curso DAW901 - Universidad Don Bosco

## 👥 Equipo de Desarrollo
- **Persona 1**: Estructura HTML y Diseño
- **Persona 2**: Lógica de Cálculos
- **Persona 3**: Sistema de Transacciones
- **Persona 4**: Formulario y Validaciones
- **Persona 5**: Tabs y Visualización

## 🚀 Tecnologías
- HTML5
- CSS3
- JavaScript Vanilla
- Bootstrap 4

## 📋 Funcionalidades
- ✅ Cálculo automático del mes actual
- ✅ Gestión de ingresos y egresos
- ✅ Cálculo de presupuesto disponible
- ✅ Porcentaje de gastos
- ✅ Validación de formularios
- ✅ Interfaz con tabs

## 🛠️ Instalación
1. Clonar el repositorio
2. Abrir `index.html` en el navegador
3. ¡Listo para usar!
